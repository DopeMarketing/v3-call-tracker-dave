# V3 Call Tracker Dave

## Overview
An internal automation tool that automatically exports Fathom call recordings to Google Drive, transcribes them, and organizes them into a searchable knowledge base. The system integrates with Google Calendar to match recordings with meeting metadata and provides basic categorization capabilities.

## Problem
Manual process of saving Fathom call recordings, transcribing them, and organizing meeting content into a searchable format is time-consuming and leads to lost insights from important conversations

## Target User
Individual professional who conducts frequent meetings and wants to automatically capture, transcribe, and organize call recordings for personal knowledge management

## Features
- Automated Fathom to Google Drive export with date/type folder structure
- Speech-to-text transcription of uploaded recordings
- Google Calendar integration for meeting metadata matching
- Manual tagging system for call categorization
- Basic search functionality across transcriptions
